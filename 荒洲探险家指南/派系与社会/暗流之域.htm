暗流之域Tribes of Shadycreek Run

被称为Tribes的敌对犯罪家族团伙维持了近一个世纪的暗流帮（Shadycreek Run）的运作，然后由于暴力内讧和内部权力斗争而失去了大部分主导地位。马尔多恩Mardoon家族、乌托洛特Uttolot家族、特雷班Trebain 家族和贾根托斯Jagentoth 家族组成的帮派们仍然控制着城市中发生的大部分业务，因为他们在暗中相互对抗。虽然每个家族中心通常由血缘或婚姻关系的亲属组成，但只要他们保持忠诚，他们的员工也被视为“家族成员”。帮派在没有统一的标志或旗帜的情况下运作，他们的利益在于走私赃物、引入和提供犯罪渠道，以及通过混乱、恐吓和恐惧来控制民众。

马尔多恩Mardoon 家族是灰败荒原Graying Wildlands的犯罪集团，这要归功于其庞大的外国联系网络以及与万象会Myriad的繁荣联盟。马尔多恩夫妇以无情的商业头脑和对可强制执行的合同的热爱而闻名，他们宁愿握手也不愿挥刀。即便如此，当合同被打破时，必须做出榜样，这种情况需要一场血腥的表演。家族财富来自妓院、珠宝和奴隶贸易，通常专门从事来自乔哈斯 Xhorhas 以及从其他地区发现的异域物品。领导层由安塞姆・马尔多恩勋爵Lord Anselm Mardoon、克里姆希尔德・马尔多恩Kriemhilde Mardoon、奥菲莉亚・马尔多恩Ophelia Mardoon、德米斯・马尔多恩Demis Mardoon和鲁弗斯・马尔多恩Rufus Mardoon共同担任。

乌托洛特Uttolot 家族通过家族贸易联盟网络而声名鹊起，使其成为帝国北部这个遥远地区最难以找到的商品进口国。它控制着通过 Quannah Breach 的大门，任何进出该地区的人都必须向 乌托洛特Uttolots 支付通行费，使他们成为其他家庭的必要伙伴，无论他们是否愿意。他们涉足偷猎危险的野兽和怪物，并经常与灰商联盟（Greytrader Union）的猎人发生冲突，灰商联盟是一群想要改善暗流之域人民生活的公民集体。与其他家族不同，乌托洛特家族的大部分武力都来自于暗流帮。米洛斯・乌托洛特勋爵Lord Milos Uttolot、米拉・乌托洛特Mira Uttolot和雅戈达・乌托洛特Jagoda Uttolot共同领导。

特雷班Trebain 家族是最隐秘的家族，他们更喜欢通过中间人和雇佣的帮手来做生意。它生活在城市边界之外，在Savalirwood森林深处，在一个围绕着要塞的宅基地网络中。特雷班家族专注于控制狩猎场和从莫莱斯迈 Molaesmyr的废墟中收集遗物。该家族的利益使其与乌托洛特 Uttolot 家族和灰商联盟 Greytrader Union 不断发生冲突。乌托洛特 Uttolots、特雷班Trebains 和灰商联盟 Greytraders 之间的三角仇恨有时会形成奇怪的联盟――毕竟，敌人的敌人就是我的朋友，无论这种联系可能多么短暂。家族的领导权由 达尔科・特雷班勋爵 Darko Trebain 、朱拉杰・特雷班Juraj Trebain、塔蒂娅・特雷班 Tatia Trebain 和 佐尔卡・特雷班Zorka Trebain 共同担任。

贾根托斯The jagentoth家族以特别残忍和无情而闻名。其成员公开处决叛徒和知名竞争对手，但承诺以丰厚回报换取绝对忠诚。该家族最成功的商业活动包括暗杀、种植和分销非法药物以及奴役（这使他们与马尔多恩 Mardoon 家族直接竞争）。领导权由 苏丽亚・贾根托斯女士 Lady Sulia jagentoth、瑞茜・贾根托斯 Reese jagentoth、努玛・贾根托斯 Numa Jagentoth 和 伊洛斯・贾根托斯 Ionos jagentoth 共同担任。

 

目标GOALS

在Shadycreek Run，家族之间的冲突是经常发生的，但万象会Myriad活动的明显缺乏，给暗流帮提供了扩大视野和影响力的独特机会。最近，帮派之间偶尔会彼此忍受足够长的时间，以便开始合资进入帝国，以重燃与万象会的旧联盟。

一些人认为，这一脆弱合作的时代是一个逃离万象会的长期阴影、并与他们平等站在一起的机会，但另一些人担心，与万象会建立新的纽带只会让它自由地从内部摧毁暗流帮。

 

关系RELATIONSHIPS

帮派互相忍耐，以便在 Shadycreek Run 原本残酷的街道上保持一定程度的文明，但他们每个人都在寻找颠覆或削弱他们的反对派、并成为城市主导家庭的方法。

所有的帮派成员都是帝国内记录在案的罪犯，并有相应的赏金。在西维南迪尔境内旅行对这些犯罪份子来说是危险的，但每个家族在帝国的主要城市中都有自己的联系人，这使商业成为可能。

在过去，万象会曾多次企图征服暗流帮，但屡遭诟病。 暗流帮对他们运转良好的邪恶引擎太自豪了，以至于无法让万象会接管。尽管暗流帮正在寻求与万象会建立新的联盟，但暗流帮的人们仍然担心这些阴暗势力之间的冲突是不可避免的。

 

重要的人物FIGURES OF INTEREST

暗流之域的生活对其帮派成员来说永远是危险的，那里的每一个微笑都隐藏着邪恶。

 

奥菲莉亚・马尔多恩OPHELIA MARDOON

中立邪恶，女性提夫林

作为 Mardoon 家族最杰出的成员之一，Ophelia 指挥着 Shadycreek Run 以北的 Sybaritic 庄园。她是个强势的享乐主义者，不喜欢被拒绝，并且拥有确保她得到她想要的任何东西的技能和人脉。作为家族中最雄心勃勃的成员之一，奥菲莉亚在灰败荒原Greying Wildlands以外的荒洲犯罪世界建立了许多联系。

 

雅戈达・乌托洛特JAGODA UTTOLOT

混乱邪恶，男性半兽人

Jagoda 是一位狩猎大师，也是研究在 Savalirwood 游荡的可怕野兽的专家，他利用自己的技能追踪和捕捉稀有生物，然后卖给他的家族。Jagoda 对他的残暴行为感到高兴，并且倾向于忽视外交关系的优雅，他偶尔也会陷入困境，而他的家族不得不把他从麻烦中解救出来。

 

达尔科・特雷班勋爵LORD DARKO TREBAIN

守序中立，男性人类

作为特雷班家族的长者，达尔科在博物馆般的豪宅中负责挖掘莫莱斯迈Molaesmyr的废墟。他痴迷于Savalirwood森林枯萎病的秘密，他确信这是由莫莱斯迈的精灵篡改了被误解的魔法造成的。他希望发现腐败的根源，并学习如何将其武器化。有了他这样的权力，特雷班的崛起将势不可挡。

 

伊洛斯・贾根托斯 lONOS JAGENTOTH

守序邪恶，男性半精灵

伊洛斯以其谈判和欺骗技巧而闻名，他已证明自己是家庭分销网络的主人。为了将贾根托斯家族的商业联盟扩展到远在 Dwendalian 帝国的北部边缘之外，伊奥诺斯一直在与万象会的特工进行安排，同时根除这些圈子中的万象会影响力。 要么绞索即将收紧，要么他设计了一个绝妙的策略来消除竞争对手。
